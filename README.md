# ВОКСЕЛИЯ

Оригинальная браузерная воксельная песочница в духе классических блочных игр.
Все текстуры, звуки, интерфейс и название — собственные, генерируются процедурно, внешние ассеты не используются.

**В комплекте две игры:**
- **Вокселия** (`index.html`) — песочница с выживанием, мобами и кооперативом
- **Блок-Страйк** (`cs.html`, кнопка «⚔ Блок-Страйк» в меню) — воксельное ПВП в духе классических тактических шутеров: карта «Дюна-2», раунды до 12 побед, 4 оружия (калаш, АВП с зумом, дигл, нож), хедшоты, боты с 3 уровнями сложности (добавление/кик в лобби), сетевая игра по коду комнаты и полная настройка клавиш

## Как запустить

Игре нужен любой локальный HTTP-сервер (ES-модули не работают через `file://`).

### Windows
Двойной клик по **`start.bat`** (нужен установленный Python или Node.js).

### Mac
1. Открой **Терминал** (Cmd+Пробел → набери «Терминал»).
2. Набери `cd ` (с пробелом) и **перетащи папку игры в окно Терминала**, нажми Enter.
3. Выполни:
```
sh start-mac.command
```
Браузер откроется сам. Если система спросит про «командные инструменты разработчика» — согласись установить (это встроенный Python) и повтори команду.

### Вручную (любая система)
```
cd путь/к/папке/игры
python3 -m http.server 8123
```
Затем открой в браузере: **http://localhost:8123**

Пока играешь, окно сервера (консоль/Терминал) должно оставаться открытым.

## Сетевая игра (кооператив)

В главном меню — «Сетевая игра»:
- **Хост**: «Создать комнату» (новый мир или продолжить свой) → в правом верхнем углу появится код комнаты (например `KJ2K`) — отправь его другу.
- **Гость**: вводит код и жмёт «Присоединиться» — мир передаётся автоматически.

Соединение идёт напрямую между игроками (WebRTC/PeerJS), свой сервер не нужен — только интернет у обоих. Хост считает мобов, дропы и время; при выходе хоста сессия завершается. Гость играет со своим рюкзаком, его прогресс в этом мире не сохраняется. В редких сетях со строгим NAT прямое соединение может не установиться — попробуйте с мобильного интернета или VPN.

## Возможности

- Процедурная генерация мира по сиду: океаны, пляжи, равнины, леса, джунгли, пустыни, горы
- Пещеры со «спагетти»-туннелями, подземными озёрами и лавовыми полостями
- Руды (уголь, железо), деревья, кусты, валуны, снежные вершины
- Цикл день/ночь: солнце, луна, звёзды, движущиеся облака, плавная смена освещения
- Погода: ясно, дождь, гроза с молниями и громом, туман; пыль в пустыне, дымка в джунглях
- Ломание и установка блоков, подсветка выбранного блока, хотбар и инвентарь
- Вода с плаванием и подводной атмосферой, лава со свечением и уроном
- Камера от 1-го и 3-го лица (F5), воксельный персонаж с анимацией
- Здоровье, урон от лавы/падения/утопления, регенерация, возрождение
- Процедурный звук: шаги по разным поверхностям, дождь, гром, вода, блоки
- Сохранение мира (сид, блоки, позиция, время) в localStorage — кнопка «Продолжить»
- Настройки: чувствительность, громкость, дальность прорисовки, качество, тени
- Оптимизация: чанки 16×80×16, отсечение скрытых граней, ambient occlusion, выгрузка дальних чанков

## Управление

| Клавиша | Действие |
|---|---|
| W A S D | движение |
| Мышь | обзор |
| Пробел | прыжок / плыть вверх |
| Shift | бег |
| Ctrl / C | присесть |
| ЛКМ (удерживать) | ломать блок |
| ПКМ | поставить блок |
| Колесо / 1–9 | выбор блока |
| E | инвентарь |
| F5 | смена камеры |
| F3 | отладка |
| Esc | пауза |

## Структура проекта

```
index.html          — разметка и экраны UI
css/style.css       — стили меню и HUD
lib/three.module.js — Three.js (локально)
js/
  main.js           — точка входа, игровой цикл, состояния
  config.js         — константы и настройки
  noise.js          — детерминированный шум (value noise, fBm)
  blocks.js         — реестр блоков, процедурные текстуры
  audio.js          — процедурный звук (WebAudio)
  sky.js            — небо, день/ночь, облака, освещение
  weather.js        — погодная система
  particles.js      — дождь, пыль, осколки, искры
  save.js           — сохранение в localStorage
  ui.js             — меню, HUD, хотбар, инвентарь
  world/
    worldgen.js     — генерация рельефа, биомов, пещер, деревьев
    chunk.js        — построение меша чанка (грани, AO)
    world.js        — менеджер чанков, блоки, рейкаст
  player/
    player.js       — физика, плавание, камеры, персонаж
    input.js        — клавиатура и мышь
```
